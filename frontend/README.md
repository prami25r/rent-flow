# RentFollow Frontend

Next.js 15 (App Router) frontend for RentFollow.

## Main folders

- `app/` – routes, layouts, and pages
- `components/` – shared UI primitives and layout components
- `features/` – domain-specific state and auth guard utilities
- `lib/` – axios client and utilities
- `types/` – API DTO and shared frontend types

## Run

```bash
npm install
npm run dev
```
